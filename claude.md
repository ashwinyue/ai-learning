# AI学习项目指南

## 📋 项目概览
本项目是一个综合性的AI学习资源集合，涵盖从基础到高级的机器学习和深度学习内容，使用Python作为主要编程语言。

## 🏗️ 项目结构
```
ai-learning/
├── week01/           # 第1周：Python基础与AI入门
│   ├── 01-python基础.py
│   ├── 02-numpy基础.py
│   ├── 03-pandas基础.py
│   ├── 04-matplotlib.py
│   └── 05-langchain.py
├── week02/           # 第2周：机器学习基础
├── week03/           # 第3周：深度学习入门
├── week04/           # 第4周：自然语言处理
├── week05/           # 第5周：计算机视觉
├── week06/           # 第6周：强化学习
├── week07/           # 第7周：AI项目实战
└── week08/           # 第8周：AI伦理与进阶
```

## 🛠️ 技术栈
- **编程语言**: Python 3.8+
- **数据处理**: NumPy, Pandas
- **可视化**: Matplotlib, Seaborn, Plotly
- **机器学习框架**: Scikit-learn, TensorFlow, PyTorch
- **自然语言处理**: NLTK, Transformers, LangChain
- **计算机视觉**: OpenCV, PIL
- **开发环境**: Jupyter Notebook, VS Code

## 🚀 快速开始

### 环境准备
1. 克隆仓库
2. 创建虚拟环境
3. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

### 运行示例
```bash
# 运行第1周示例
python week01/01-python基础.py

# 使用Jupyter运行
jupyter notebook week01/
```

## ⚙️ 开发规范

### 代码标准
- 遵循PEP 8编码规范
- 使用描述性的中文命名函数和变量
- 包含全面的注释和文档字符串
- 每个示例都应该是自包含的

### 数据管理
- 使用相对路径引用数据文件
- 数据集存储在`data/`目录中
- 大文件使用Git LFS管理

### 依赖管理
- 所有依赖项列在`requirements.txt`中
- 固定具体版本以避免兼容性问题
- 定期更新依赖项

## 🐛 常见问题与解决方案

### 环境问题
- **Python版本**: 使用Python 3.8-3.10
- **包冲突**: 使用conda或venv进行环境隔离
- **CUDA问题**: GPU用户安装匹配的CUDA和cuDNN版本

### 运行问题
- **缺少依赖**: 运行`pip install -r requirements.txt`
- **路径错误**: 检查相对路径
- **内存不足**: 减小批次大小或使用CPU
